Introduction

Lily is very light weight resposive framework.It is written is sass and easy to configure.Even you can change class names at ease.

Setup

Clone repo from git.

Execute cmd grunt watch.

If you don't want to configure names and number of columns just include lily.css in your project.

Link to download

Configurations

					
$globel-prefixes:webkit moz ms o;
$normalize:false;

/*grid configurations*/
$col-name:span;
$col-lg-prefix:span-lg-;
$col-md-prefix:span-md-;
$col-sm-prefix:span-sm-;

$col-lg-lower-limit:992px;
$col-md-lower-limit:768px;

$gutter-width:0%;
$col-padding:0%;
$no-of-columns:16;

Currently we offer two breakpoints.

By changing prefixes you can change class names.

Example

The Lily block is written using 16 lily grid.

					
<div class="wrapper">
	<div class="row">
		<div class="span span-lg-2 span-md-4 span-sm-8"> 2 - column </div>	
		<div class="span span-lg-2 span-md-4 span-sm-8"> 2 - column </div>	
		<div class="span span-lg-4 span-md-8 span-sm-16"> 4 - column </div>	
		<div class="span span-lg-8 span-md-16 span-sm-16"> 8 - column </div>
	</div>			
</div>

Other classes

wrapper-fluid - container without padding

wrapper - container with 5% padding

clearfix - clear floats


visible-lg - visible only in large deivces

visible-md - visible only in medium deivces

visible-sm - visible only in small deivces


hidden-lg - hidden in large devices

hidden-md - hidden in medium devices

hidden-sm - hidden in small devices

(*Note : don't use inapporopriate options ex class="visible-lg hidden-lg")
